## Installation

How to install
```bash
1. git clone https://github.com/lainsec/rchan
2. cd rchan
3. python3 -m venv rchanenv
4. source rchanenv/bin/activate
5. pip install -r requirements.txt
```
How to use
```bash
python3 app.py
`The first account will automatically recive OWNER permissions.`
```
